Personal Habit Tracker Suite

A 3-part Java backend development suite progressing from low-level database drivers to enterprise web applications.

💻 Project 1: JDBC Console Habit Checklist
A CLI application managing daily goals directly inside a local MySQL database.

Tech Stack: Java SE (Scanner UI), JDBC API, Data Access Layer (DAL) Architecture.

Key Concept: Managing explicit database lifecycles and using PreparedStatement to block SQL injection.

Quick Run: Configure your MySQL credentials in dal.HabitDAL ➔ Run main.MainApp as a Java Application.

🌐 Project 2: Servlet & JSP Web Dashboard
An interactive, browser-based MVC web application that tracks habit completion in real-time.

Tech Stack: Java EE (Servlets, JSPs), Tomcat Server, MVC Design Pattern.

Key Concept: Enterprise request routing, server-side data dynamic rendering, and deployment descriptor (web.xml) mapping.

Quick Run: Right-click project folder ➔ Run As ➔ Run on Server (Tomcat v9.0+) ➔ Navigate to /Servlet_JSP_Web_Dashboard/habits.

📋 Global Setup Requirements
Environment: Java JDK 17+ & MySQL Server

Server: Apache Tomcat v9.0+

Dependency: MySQL Connector/J Driver added to Build Path
